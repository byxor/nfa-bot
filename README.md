# nfa-bot

A discord bot for the THPS community in NFA (No Fun Allowed)

### To run the bot:

`./gradlew clean run`

### To run the tests:

`./gradlew clean test`

### Current features:

#### Ping pong

If you say "Ping!", the bot will say "Pong!". This is purely an experimental feature to get some of the architecture set up.
